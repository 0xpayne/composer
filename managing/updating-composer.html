Updating Hyperledger Composer

After deploying Hyperledger Composer you may wish to upgrade to a new version. To update your installed version of Hyperledger Composer you must uninstall the client, admin, and runtime CLI components and reinstall them by using npm.

Procedure

  1. Uninstall the Hyperledger Composer components by using the following commands:

    npm uninstall -g composer-cli
npm uninstall -g composer-rest-server
npm uninstall -g generator-fabric-composer

  2. Install the latest version of the Hyperledger Composer components by using the following commands:

    npm install -g composer-cli
npm install -g composer-rest-server
npm install -g generator-fabric-composer
