Installing Hyperledger Composer

There are two ways to install Composer. To model, develop, or test business networks, install the Composer Playground locally or use the playground online.

If you want to develop and test applications, connect to a running Hyperledger Fabric instance, or run Hyperledger Composer and Hyperledger Fabric locally, see installing Hyperledger Composer.

If you want a more guided "Getting Started" experience, see our Tutorials and Getting Started guides.

After installing, create a connection profile to connect to a running Hyperledger Fabric instance. See creating a connection profile
