Integrating with Node-RED

Node-RED is a lightweight Open Source integration technology, written in JavaScript. It uses a graphical flow to integrate different nodes, where nodes can receive data, transform data and output data.

Node-RED is commonly used to rapidly prototype Internet of Things style applications, or to wire existing Internet services together. You can use the Hyperledger Composer Node-RED contribution to:

  • Submit transactions
  • Read and update assets and participants
  • Subscribe to events
  • Delete assets and participants

The Hyperledger Composer Node-RED nodes are distributed as a standalone npm package, published here: - https://www.npmjs.com/package/node-red-contrib-hyperledger-composer
